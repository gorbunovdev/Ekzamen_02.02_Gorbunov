Телеком-Калькулятор
Программа для расчета стоимости телефонных услуг и генерации чеков.

Возможности
Расчет по двум тарифам (200/100 мин лимита)

Генерация квитанций в формате Чек_№_дата_ФИО.txt

Валидация ввода данных

Использование
Введите ФИО и минуты

Выберите тариф (1 или 2)

Нажмите "Рассчитать"

Создайте квитанцию кнопкой "Создать квитанцию"

Формула расчета
В пределах лимита: минуты × базовая ставка

Сверх лимита: лимит × базовая ставка + сверхлимит × 1.6 руб.

Технически
WPF приложение на C#

Сохраняет чеки в выбранную папку

Автонумерация чеков


Диаграммы находятся в папке "Диаграммы"
